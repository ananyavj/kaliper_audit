Segment webhook
Go to app.amplitude.com → your project → Settings → Projects → [your project]. Copy the API Key and Secret Key from there.
Copy this webhook URL and paste it into Segment → Connections → Destinations → Add Destination → Webhooks (Actions). Segment will forward every event to Kaliper automatically.
In Segment:
1. Go to Connections → Destinations → Add Destination
2. Search for "Webhooks (Actions)" and select it
3. Connect it to your source, name it "Kaliper"
4. In Mappings, add a new mapping → Send → Event Type is Track
5. Set the URL field to the webhook URL above
6. Enable the destination
